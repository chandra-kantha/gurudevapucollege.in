# Gurudeva College Website

## Description
This is the official website for Gurudeva College, located in Belthangady, Dakshina Kannada, Karnataka. It showcases information about the Pre University College and First Grade College, including news, courses, facilities, and more.

## Features
- Responsive design with Bootstrap
- Image sliders for college highlights
- News and events sections
- Contact and course information
- Gallery and staff details
- Integrated Google Analytics and Firebase

## Setup
1. Clone or download the project to your local machine.
2. Open `index.html` in a web browser to view the site.
3. For local development, use a simple HTTP server (e.g., Python: `python -m http.server` in the project directory).

## Technologies Used
- HTML5
- CSS3 (with Bootstrap, Animate.css, etc.)
- JavaScript (jQuery, Owl Carousel, etc.)
- Firebase for potential backend features
